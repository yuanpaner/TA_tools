JJTools8

The scripts contained in this file were written for 3DStudio Max R8.  They should work fine in R5, R6 and R7 but they have not been rigourously tested.  Please download R4, R5, R6 or R7 scripts from www.jimjagger.com

Installation
Place all .mcr files in 3dsmax8\ui\macroscripts
Place all .bmp files in 3dsmax8\ui\icons
Place JJTools8_Functions.ms in 3dsmax8\stdplugs\stdscripts
Place JJTools8_SelectOpposite.ms in 3dsmax8\Scripts\JJTools8
Place JJTools8_SelectReloadfile.ms in 3dsmax8\Scripts\startup

Restart Max
JJTools8 can be found in the Customise UI window under the category JJTools 8
